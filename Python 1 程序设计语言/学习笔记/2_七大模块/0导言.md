## Python计算生态

**Python计算生态 = 标准库 + 第三方库**

- 标准库：随解释器直接安装到操作系统中的功能模块
- 第三方库：需要经过安装才能使用的功能模块

库 `Library`、包 `Package`、模块 `Module`，统称**模块**。
